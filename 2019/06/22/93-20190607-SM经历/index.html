【SM经历】2019/06/07 珠海の行

因为一场调教，我们选择了互相信任；因为一纸契约，我们走在了一起；因为一种关系，走进了彼此的内心

S：麵包師「AnPan@S」
M：QIU QIU
场地：@ 珠海
体验过程：7号家具 + SP + 后手直臂缚 + M字缚，8号跪缚 + SP + 2TK + 水刑 + 窒息

QIU QIU の体验感受：

上周五我们一家人去了珠海游

也因此有了暂时和宝宝分开，与主人单独相处的二人世界的机会，主人当然不会放过这么好的机会，肯定是好好的虐我一把了。

主人事先帮我准备了一套黑色性感睡衣加吊带袜，还有一条丝带。因为我牙齿的关系，主人放弃了他最喜欢的口腔堵塞，这一点让我很感动。
当我换完衣服后，主人就开始对我发号施令了，以后就是一系列的捆绑，各种变换姿势加调教，因为不在家，没有孩子在旁边，所以感觉格外放的开，玩的很嗨。直到后来主人背插到一半后又突然增加了一套认真的裸缚，这个过程让我很意外，身体本来做好了最后冲刺得准备，谁料又要重新回到原点，再者因为紧缚时间已经长达两小时，来回变换缚式让我的身体有了很重的疲累感，所以这一段过程我是抗拒的，很抵触，我觉得主人把我当成了玩具，我伤心的哭了，也包含着平常日子里的疲累，生活工作中的不如意都放在其中发泄了出来。

调教期间，主人用皮鞭惩罚我，原因是我最近一段时间没有按规定向主人请安…通过这次教育，我记住了以后要早晚按规矩请安，没有规矩不成方圆的道理！

再说下第二天早上的调教，这个是我万万没想到的，前一晚折腾那么厉害，第二天早上一大早又来…而且这次主人竟然拉开了窗帘，只留一层薄纱，马路对面的楼里应该能看清房间里的人…并且这次主人边爱边拍摄裸照，还内射不带套，接二连三的不在我接受范围内的事情，成为压垮骆驼的最后一根稻草…我和主人赌气了。我觉得他没有尊重我，也没有遵守以前的承诺。
其实在爱爱途中，这些我都知道也都很抗拒，但身为小奴，应该无条件服从，因此我选择了事后的赌气而不是中途喊停。也希望主人能理解小奴，给予小奴相应的安全感。

ANPAN@S の思考

我最近有在思考一个问题，自己后续的发展重点是哪个方向

经历了这一年的对外交流，慢慢感觉什么方面都涉猎大而全，但是不够精与专，这不是我的追求，并且明显感觉力不从心，做一件事情，就一定要做好，经过这段时间的考虑，我决定选择后者，我想要在某一方面做的专业，其他方面有了解就足够，因此我确定走绳艺这条路，对“绳子”的执念，可能对于现在的我，以及你我的主奴契约、原则、底线，可能更为合适，绳缚+艺术，一切的交流都是基于“绳子”，以绳子为中心，绳缚的造型+拍摄+连接+绳责，这是我从今天开始的唯一方向。

字母圈的内容太过丰富，让我有些应接不暇，什么都想做，什么都能做，结果什么都做的不伦不类，同时也必然会有太多的诱惑与深坑，字母圈不能脱离现实生活，也不可能与事实伦理背道而驰。就像我不敢将自己这一爱好告诉现实生活中的亲朋好友，现在的我还没有这个勇气与胆量，因为我觉得虽然这个社会越来越开放话，人们的接受绳缚越来越高了，但是还没有到真正能接受字母和绳艺这种，被称为所谓的另类与变态的程度。

所以，我只能小心的追求着自己这一“另类”的爱好，这个爱好是与生俱来，没有原因，是我的一种最为真实的本性体现，在那个状态下，才是最最真实的自己。很多时候，人都会伪装自己，将自己真实的本能的欲望不愿意过多的表现出来，很多时候是因为这个社会的压力，以及人与人之间的不信任，以及整个环境的压迫所导致的，所以，为了给自己一个展示真实内心的机会与空间，我选择了进入所谓的“圈子”，理解圈子的文化，与圈子内的朋友交流沟通，这些行为最终的目的是什么呢？难道就是为了互相平时发发照片，让别人知道你的存在，让别人赞美一下自己玩的技术有多么牛逼轰轰，给自己的虚荣心增加一些色彩？这些都是很表面的东西，真正有什么涵养吗？我现在已经对这些形成了一种“特殊”的免疫，QQ与微信在群里的聊天越来越少，很多时候对聊天内容都没有兴趣去看了，这可能就是对目前状态的一种“麻木”态度。

这种状态是一种消极的表现形式，我必须要改变，所以，我要通过“绳子”改变我目前的这种状态，调整心态，继续前行。

露而不裸，妖而不骚，以一种纯粹的眼光去理解与看待“绳艺”的作品，去体会与绳子的结合，柔虐、凄美、欲孽。
